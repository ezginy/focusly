# Focusly ✨

A modern productivity dashboard built with Vanilla JavaScript.

Focusly is a responsive Pomodoro and task management application designed to help users stay focused, organized, and consistent throughout the day.

This project was developed as part of my frontend learning journey while studying Software Development.

---

## Features

- Multi-page dashboard navigation
- Pomodoro focus timer
- Task management system
- Session analytics and progress tracking
- Dark / Light theme support
- Responsive layout for mobile devices
- LocalStorage persistence
- Dynamic page titles
- Smooth transitions and UI interactions
- Accessibility-focused UI improvements

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
